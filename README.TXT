PROJECT TITLE: Checkers GUI App
VERSION or DATE: Created 2019; Rehashed 2021
AUTHORS: Abhisar Mittal
ABOUT THE PROJECT: The app is a two-user checkers app that can be played on a local machine.
                    It makes use of Java GUI (Graphical User Interface).
                    The algorithms and algorithmic design can be found in the CheckersLogic class (CheckersLogic.java).
                    Other classes are ancillary.

USER INSTRUCTIONS:
Clone the repository on a local machine.
Note that CheckersRunner.java is a runner class.
Run the Main function on a Java SDK.
The app (GUI output) opens in a separate window on the local machine.
